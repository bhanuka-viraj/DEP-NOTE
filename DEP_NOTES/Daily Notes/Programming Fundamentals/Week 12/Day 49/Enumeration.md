How does this Enum works in internally 

```Java
enum Month {  
    JAN("January"),  
    FEB("February"),  
    MAR("March"),  
    APR("April"),  
    MAY("May"),  
    AUG("June"),  
    SEP("September"),  
    OCT("October"),  
    NOV("November"),  
    DEC("December");  
  
    private String fullName;  
  
    public String fullName() {  
        return fullName;  
    }  
  
    private Month(String fullName) {  
        this.fullName = fullName;  
    }  
}
```
